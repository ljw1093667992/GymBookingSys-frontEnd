
### 注册

**请求URL：**
- /user
**请求方式：**
- POST

**参数：**

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|account |是  | 账号   |
|password |是  | |
|role |是  | 0为普通用户，1为管理员用户|

**返回示例**
密码长度需要[5, 18]，account需要 [5, 25]
 ```
 {
    "code": 1,
    "msg": "success",
 }
 {
    "code": 0,
    "msg": "fail",
 }
 ```
